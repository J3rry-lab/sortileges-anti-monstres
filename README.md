# 🔮 Générateur de Sortilèges Anti-Monstres Modernes

Un **générateur de sortilèges aléatoires** pour combattre les monstres modernes : trolls, algorithmes oppressifs, bugs malveillants, et autres nuisances du web.

Développé avec **Flask et Python**, ce projet est une **arme poétique** contre la toxicité en ligne.

🐱 **Avec la bénédiction de Paco, Gardien Suprême du Clavier.**

---

## ✨ Fonctionnalités
- Génération aléatoire de sortilèges avec :
  - Un **nom** (ex : "L’Exorcisme des Trolls").
  - Un **ingrédient secret** (ex : "une larme de café froid").
  - Un **effet magique** (ex : "transforme les haters en statues de sel").
- Compteur de sortilèges lancés.
- Page "À propos" pour expliquer le projet.
